Enter the name for the output disk image file. You do not need to add .adf etc, Disk2FDI will add it for you. Enter the full path to the desired location if you don't want the image created in ADF Opus' directory.

Examples:
Entering "myimage" with ADF selected will result in a disk image named myimage.adf being created in the same directory as Disk2FDI.
Entering "C:\Images\myimage" with ADF selected will create a disk image named myimage.adf in C:\Images.
