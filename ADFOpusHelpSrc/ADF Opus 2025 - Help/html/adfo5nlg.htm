The Create New Volume Dialogue

Click on parts of the dialogue for help.

Usage

Use this dialogue to create a new, empty disk image or hardfile. You can select the type, capacity, file system flags and disk label for your creation using the available options. Selecting the "Open after creating" option causes ADF Opus to open the new image in a new child window after it has been created.

Use the Batch Converter to convert your new ADF to an ADZ, if you want to compress it.
