Known Bugs/Limitations

V1.0

ADFLib doesn't recognise hardfiles smaller than 1802240 bytes (1.7 MB). This isn't an

issue as you would expect a hardfile to be larger than a floppy disk, anyway.

V1.2

End of line markers show up in the text viewer as an ugly vertical black bar. I have

spent hours trying to trace the cause of this, even using MS example code and can't

find it yet.
